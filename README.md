# Tweets-Database-Manager

## Install

Prerequisite:
mysql installed, or alternative database

```
cd Tweets-Database-Manager/
python3 -m venv venv  
source venv/bin/activate
# upgrade your pip, make sure pip -V is 18.1, otherwise, update
pip install -r requirements.txt
```
